# Welcome to FOS Documentation

FOS is a comprehensive blockchain platform designed for **NFT minting**, **whitelist verification**, and **prize distribution**. This documentation provides everything you need to understand, integrate, and build with our platform.

## What is FOS?

FOS is a modular blockchain ecosystem that enables:

- **Free and Whitelist Minting** - Flexible minting mechanics with Merkle-tree based verification
- **Prize Distribution** - On-chain and hybrid prize allocation with verifiable randomness
- **Secure Smart Contracts** - Audited contracts following industry standards (ERC-721/1155)
- **Modern Web Platform** - Next-gen landing and application interfaces

## System Architecture

The FOS system comprises three main components:

**On-Chain Layer:**
- Smart Contracts for token/NFT logic
- Whitelist Verification
- Prize Distribution

**Off-Chain Layer:**
- Backend APIs
- Indexers (The Graph)
- Merkle Service

**Frontend Layer:**
- Landing Website
- Main Application
- Admin Dashboard
