Comprehensive infrastructure documentation for the FOS platform.

## CI/CD Pipeline

### GitHub Actions

```yaml
# .github/workflows/deploy.yml
name: Deploy
on:
  push:
    branches: [main, develop]

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: '20'
      - run: npm ci
      - run: npm run lint
      - run: npm run test
      - run: npm run build
      - uses: vercel/actions@v1
        with:
          vercel-token: ${{ secrets.VERCEL_TOKEN }}
```

## Security

### Key Management

- **Development**: Local environment variables
- **Staging/Production**: Vercel encrypted secrets
- **Smart Contract Admin**: Hardware wallet (Ledger)
- **Hot Wallets**: Server-side with KMS

### Access Control

| Role | Permissions |
|------|-------------|
| **Developer** | Read logs, deploy to staging |
| **DevOps** | All deployments, secrets management |
| **Admin** | Contract admin operations |
